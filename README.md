## Overview

Systems-University is a curated learning and experimentation repository documenting my journey through systems programming, Linux internals, cybersecurity, backend engineering, and low-level computing concepts.

This repository is designed to emphasize practical understanding through:
- hands-on experimentation
- Linux-based development
- systems-level problem solving
- security labs
- networking exercises
- containerized environments
- backend infrastructure projects
- CTF-style challenges

The goal is not only to learn how systems work, but to understand how they fail, how they can be secured, and how they can be engineered efficiently.

## Featured Labs & Experiments

### Linux Shenanigans
Hands-on privilege escalation, filesystem abuse, symlink attacks, shell startup injection, and Linux permission exploitation experiments.

### Dockerized Security Labs
Containerized Linux environments designed to simulate realistic security and systems scenarios.

### Bash Automation
Shell scripting projects focused on automation, process handling, workflow optimization, and systems interaction.

### Networking Experiments
Practical exploration of networking concepts including sockets, packet analysis, communication protocols, and network troubleshooting.

### Operating Systems Exploration
Notes, experiments, and implementations related to memory management, process scheduling, paging algorithms, and low-level OS concepts.

### Cybersecurity Challenges
CTF-style exercises and security research focused on Linux exploitation, privilege escalation, environment manipulation, and defensive concepts.

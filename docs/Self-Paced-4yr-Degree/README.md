# Four-Year Self-Paced Computer Science Degree

A rigorous, university-level Computer Science education built through self-study, project-based learning, and open educational resources from Stanford, MIT, Carnegie Mellon, Princeton, and other leading institutions.

## Mission

The purpose of this project is to replicate the depth, rigor, and breadth of a traditional Bachelor of Science in Computer Science while emphasizing hands-on engineering and public documentation.

Rather than focusing solely on completing courses, the goal is to develop a deep understanding of computer science fundamentals, systems engineering, software development, and security through a combination of:

* Structured coursework
* Mathematical foundations
* Laboratory assignments
* Independent research
* Large-scale software projects
* Open-source contributions

Every concept studied is accompanied by implementation, documentation, and reflection. The repository serves as both a learning journal and a verifiable record of progress.

## Program Overview

This curriculum spans four years and approximately 120+ credit hours, covering:

### Core Computer Science

* Programming Methodology
* Programming Abstractions
* Discrete Mathematics
* Data Structures
* Algorithms
* Probability for Computer Scientists
* Computer Organization
* Operating Systems
* Computer Networks
* Compilers

### Mathematics & Science

* Calculus I & II
* Linear Algebra
* Multivariable Calculus
* Physics I & II

### Systems & Security

* Computer Systems
* Computer and Network Security
* Computer Systems Security
* Advanced Data Structures
* Nand2Tetris
* Operating System Development

### Capstone

The final year culminates in the design, implementation, documentation, and presentation of a substantial systems or security-focused project.

## Learning Philosophy

This degree follows several guiding principles:

1. Learn fundamentals before frameworks.
2. Build everything possible from first principles.
3. Document understanding through writing.
4. Prioritize projects over passive consumption.
5. Maintain consistency over intensity.
6. Share progress publicly and transparently.

## Expected Outcomes

By the completion of this program, I aim to:

* Master the core concepts of Computer Science.
* Build complex software systems from scratch.
* Understand modern operating systems and networks.
* Read and analyze academic literature.
* Contribute to open-source projects.
* Develop a portfolio demonstrating practical engineering ability.
* Possess knowledge comparable to a traditional Computer Science graduate.

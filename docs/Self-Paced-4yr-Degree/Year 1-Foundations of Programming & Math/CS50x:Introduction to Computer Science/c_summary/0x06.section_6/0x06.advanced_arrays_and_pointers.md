## Advanced C Arrays
What:
    - Elements layout contiguosly in memory.
Refer to elment:
    - Using square brackets.
    - Refer by address of first element ie "base address"
```
{
    int array[6];

```
- Address of nth element:
    ```
    address of nth element = address_of_0th_element + (n * element_size_in_bytes)
    ```
- [] syntax deals with arithmetic.

## '+' Syntax
- A + pointer = pointer

- [] give nth element, + gives pointer to nth element.

- intArray[3] equivalent to `*(intArray + 3)`.

## Pointer++ Style -- strcpy()
- p pointer to an element, (p+1) points to next element in array. Use p++ to step to pointer over the element in array.

- Implementation of strcpy
```
void strcpy1(char dest[], const char source[])
{
    int i = 0;

    while (1)
    {
        dest[i] = source[i];
        if (dest[i] == '\0') break;     // we're done
        i++;
    }
}

// Move assignment into the test
void strcpy2(cahr dest [], const char source[])
{
    int i = 0;

    while (dest[i] = source[i]) != '\0')
    {
        i++;
    }
}

// Get rid of i and just move the pointers.
// Relies on the precedence of * and ++.
void strcpy3(char dest[], const char source[])
{
    while ((*dest++ = *source++) != '\0');
}

// Rely on the fact that '\0' is equivalent to FALSE
void strcpy4(char dest[], const char source[])
{
    while (*dest++ = *source++) ;
}
```

## Array Names Are Const
- Base address of array can't be changed.
```
{
    int ints[100];
    int *p;
    int i;

    ints = NULL;        // NO, cannot change the base addr ptr
    ints = &i;          // NO
    ints = ints + 1;    // NO
    ints++;             // NO

    p = ints;       // OK, p is a regular pointer which can be changed
                    // here it is getting a copy of the ints pointer

    p++;            // OK, p can still be changed (and ints cannot)
    p = NULL;       // OK
    p = &i;         // OK

    foo(ints);      // OK (possible foo definitions are below)
}
```
- Array parameters are passed as pointers.
```
void foo(int arrayParam[])
{
    arrayParam = NULL;          // Silly but valid, Just changes the local pointer
}

void foo(int *arrayParam)
{
    arrayParam = NULL;      //ditto
}
```
## Heap Memory
- Prototypes are in `<stdlib.h>`, Code that wants to use these must #include <stdlib.h>.

- Functions include:
```
    void* malloc(size_t size);
        - Request contiguous block of memory from heap, return pointer to block otherwise, NULL.
        - **size_t** - Unsigned long, measured in bytes.

    void free(void* block);
        - Take pointer to heap block allocated by malloc, and returns the block to the heap for re-use.
        - Note:
            - Don't use after free or double free.

    void* realloc(void* block, size_t size); ie resize block.
        - Resize block of heap memory.
        - Takes care of moving the bytes from the old block to the new block pointer.
```
## Memory Management
- free() memory allocated by malloc and realloc

## Dynamic Arrays
- Allocating array in heap.
```
{
    int a[1000];

    int *b;
    b = (int*) malloc( sizeof(int) * 1000);
    assert(b != NULL);          // Check that the allocation succeeded.

    a[123] = 13;        // Just use good ol' [] to access elements
    b[123] = 13;        // in both arrays.

    free(b);
}
```
## Advantages of being in the heap
    - Define size at runtime.
    - Exists until deallocated with free()
    - Runtime size change using `realloc()`.
    ```
    ...
    b = realloc(b, size(int) * 2000);
    assert(b != NULL);
    ```
## Disadvantages of being in the heap
    - You have to remember and get right, what:
        - Allocation
        - Deallocation
    - If get above wrong, have runtime crashes.

## Dynamic Strings
- Dynamic allocation of arrays works well for allocating strings in the heap.
```
#include <string.h>

/*
   Takes a c string as input and makes a copy of that string in the heap. the caller takes over ownership of the new string and is repsonsible for freeing it.
*/
char *MakeStringInHeap(const char *source)
{
    char *newString;

    newString = (char*) malloc(strlen(source) + 1);  // +1 for the '\0'
    assert(newString != NULL);
    strcpy(newString, source);
    return(newString);
}
```

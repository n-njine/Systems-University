# Details and Library Functions

## Precedence and Associativity
```
    function-call() [] -> .                             L to R

    ! ~ ++ -- + - *(ptr deref) sizeof &(addr of)      **R to L**

    * / %                                               L to R
    (the top tier arithmetic binary ops)

    + -                                                 L to R
    (second tier arithmetic binary ops)

    < <= > >=                                           L to R

    == !=                                               L to R

    in order: & ^ | && ||                               L to R

    = and all its variants                            **R to L**

    , (comma) .                                         L to R

    Parenthesis must:
        *structptr.field   NO
        (*structptr).field or structptr->field
```

## Standard Library Functions
- To use these, #include appropriate .h file
    - stdio.h       file input and output
    - ctype.h       character tests
    - string.h      string operations
    - math.h        mathematical functions such as sin() and cos()
    - stdlib.h      utitlity functions such as malloc and rand()
    - assert.h      the assert() debugging macro
    - stdarg.h      support for functions with variable number of arguments
    - setjmp.h      support for non-local flow control jumps
    - signal.h      support for exceptional condition signals
    - time.h        date and time
    - limits.h, float.h     constants which define type range values such as `INT_MAX`.

### stdio.h
Have, functions that:
    - Print and read strings
    - Open and close files on file system.
---
    FILE* fopen(const char* fname, const char* mode);
        - MOde = "r" read, "w" write, "a" append. return NULL on error.

    int fclose(FILE* file);
        - Closes a file. Returns EOF on error.

    int fgetc(FILE* in);
        - Read and return next unsigned char out of a file. or EOF if the file has been exhausted.

    char* fgets(char* dest, int n, FILE* in)
        - Reads line text upto '\n' character. The '\n' is included in the string. Returns NULL on EOF or error.

    int fputc(int ch, FILE* out);
        - Write the char to the file as an unsigned char. Returns ch, or EOF on err.

    int ungetc(int ch, FILE* in);
        - Push the most recent fgetc() char back onto the file. Returns ch or EOF on error.

    int printf(const char* format_stirng, ...);
        - Prints a string with values possibly inserted into it to standard ouput.
        - Takes format string followed by matching arguments. Format string has text mixed with % directives marking things to be inserted in the ouput.

    int scanf(const char* format, ...)
        - Read characters from standard input trying to match elements in the format string.
        - Each percent directive  in the format string must have a matching pointer int the argument list which scanf() uses to store the values it finds.
        - Skips whitespace as it tries to read in each percent directive.
        -  Returns the number of percent directives processed successfully.
        Example:
        {
            int num;
            char s1[1000];
            char s2[1000];

            scanf("hello %d %s %s", &num, s1, s2);
        }
        - Looks for the word "hello" followed by a number and two words.
        - Uses poiters &num, s1, and s2 to store what it finds into the local variables.
---

### ctype.h
- Includes macros for doing simple tests and operations on characters.
```
    isalpha(ch)                 // ch is an upper or lowercase letter

    islower(ch), isupper(ch)    // Same as above, but upper/lower specific

    isspace(ch)                 // ch is a whitespace character such as tab, space, newline, etc

    isdigit(ch)                 // digit such as '0'..'9'

    toupper(ch), tolower(ch)    // Return the lower or upper case version of a alphabetic characer, otherwise pass it through unchanged
```
### string.h
- Caller must ensure there's enough memory for the routines.
- The type `size_t` is unsigned wide enough for computer's address space.
```
    size_t strlen(const char* string);
        Return the number of chars in a C string. 

    char* strcpy(char* dest, const char* source);
        Copy the characters from the source string to the destination string.

    size_t strlcpy(char* dest, const char* source, size_t dest_size);
        Like strcpy(), knows the size of the dest, use to avoid memory errors and buffer-overflow security problems.

    char *strcat(char* dest, const char* source);
        Append the characters from the source string to the end of destination string.

    int strcmp(const char* a, const char* b);
        Compare two strings and return an int which encodes their ordering:
            zero 0 : a == b
            negative: a < b
            positive: a > b

    char* strchr(const char* searchIn, char ch);
        Search the given string for the first occurence of the given character.
        Returns a pointer to the character, or NULL if none is found.

    char* strstr(const char* searchIn, const char* searchFor);
        Similar to strch(), but searches for an entire string instead of a single character. Search's case sensitive.

    void* memcpy(void* dest, const void* source, size_t n);
        Copy the given number of bytes from the source to the destination. The source and destination must not overlap.

    void* memmove(void* dest, const void* source, size_t n);
        Similar to memcpy() allows areas to overlap.
```
### stdlib.h
```
    int rand();
        Return pseudo random integer in the range 0..RAND_MAX(limits.h), that's at least
        32767

    void srand(unsigned int seed);
        Same as rand() but sets seed

    void* malloc(size_t size);
        Allocate a heap block of the given size in bytes.

    void free(void*  block);
        Returns a previous malloc block to the system for reuse.

    void* realloc(void* block, size_t size);
        Resize an existing block to the new size. Returns NULL if the resize operation was not possible.

    void exit(int status);
        Halt and exit the program and pass a condition int back to the operating system.

    void* bsearch(const void* key, const void* base, size_t elem_size, <compare_function>);
        Do a binary search in an array of elements.

    void qsort(void* base, size_t len, size_t elem_size, <compare_function>);
        Sort an array of elements.

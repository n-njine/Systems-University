# Create a Python 3.14 Environment
---
Step 1: Create a Python 3.14 Environment
---

Use anaconda to create a virtual environment for python3.14
```
conda create -n CS106A python=3.14 -c conda-forge
```

---
Step 2: Activate the Environment
---
```
conda activate CS106A
```

---
Step 3: Connect it to VS Code
---
1. Open your assingment folder in VS Code.
2. Press `Ctrl + Shift + P` to open the Command Palette.
3. Type `Python: Select Interpreter.`
4. Look for the option that says `Python 3.14x ('CS106A': conda) and click it.

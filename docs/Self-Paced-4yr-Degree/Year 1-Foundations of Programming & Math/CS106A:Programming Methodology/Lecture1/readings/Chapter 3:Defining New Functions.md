# Chapter 3:Defining New Functions

## Defining New Commands
You do so by defining new commands:
```
def name():
    *commands that make up the body of the functions*
```
name: Name you have chosen for the new function.

sequence of commands are provided after the name that are indented by one tab.

E.g:
```
def turn_right():
    turn_left()
    turn_left()
    turn_left()

# Another one
def turn_around():
    turn_left()
    turn_left()
```

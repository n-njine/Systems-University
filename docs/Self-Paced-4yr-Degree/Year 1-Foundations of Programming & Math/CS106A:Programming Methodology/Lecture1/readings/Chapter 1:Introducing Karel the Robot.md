Karel is a programming environment that teaches the basic ideas of programming that removes the complexities that characterize programming languages.

## What is Karel?
Karel is a simple robot living in a simple world. You can give Karel a set of commands, that asks Karel to perform certain tasks. The act of giving these instructions is what we call *programming*.

### Karel's world.
Karel's world is defined by rows running horizontally (east-west) and colums running vertically (north-south).

The intersection of a row and a column is called a corner.Karel can only be positioned on corners and must be facing one of the four compass directions (north, south, east, west).
